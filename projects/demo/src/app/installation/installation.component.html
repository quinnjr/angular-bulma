Installation

Getting you up and running with @angular-bulma libraries

Getting started with each @angular-bulma is easy!

First, install the @angular-bulma library of your choice using your preferred nodejs package manager:

        
npm i --save @angular-bulma/components
        
      
        
yarn add @angular-bulma/components
        
      
        
pnpm add @angular-bulma/components

@angular-bulma libraries do NOT come with Bulma stylings automatically loaded. You'll need to import Bulma's CSS into your project in styles.scss:

        
@import '~bulma';

Alternatively, you can load Bulma's CSS from your prefered CDN service in your index.html file:

        
<head>
...
  <link
    rel="stylesheet"
    href="https://cdnjs.cloudflare.com/ajax/libs/bulma/0.9.3/css/bulma.min.css"
    integrity="sha512-IgmDkwzs96t4SrChW29No3NXBIBv8baW490zk5aXvhCD8vuZM3yUSkbyTBcXohkySecyzIrUwiF/qV0cuPcL3Q=="
    crossorigin="anonymous"
    referrerpolicy="no-referrer"
  />
</head>

Finally, import your desired modules into your Angular project:

        
...
import { BulmaComponentsModule } from '@angular-bulma/components';

@NgModule({
  imports: [
    BrowserModule,
    AppRoutingModule,
    // Angular-Bulma modules
    BulmaComponentsModule
  ],
  declarations: [AppComponent],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule {}
