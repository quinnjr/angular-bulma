Flexbox Directives

Helpers for all Flexbox properties

Combined with the bu-is-flex directive, all of the Flexbox CSS properties are available as @angular-bulma/directives directives:
  • flex-direction
  • flex-wrap
  • justify-content
  • align-content
  • align-items
  • align-self
  • flex-grow
  • flex-shrink
