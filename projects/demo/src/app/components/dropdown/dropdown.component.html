Dropdown

An interactive dropdown menu for discoverable content.

A bu-dropwdown component is a container for a dropdown button and a dropdown menu:

  • bu-dropdown is the main container
  • All content defined inside the bu-dropdown is either a dropdown-item or dropdown-divider

# Hoverable or Togglable

The bu-dropdown component has 2 additional modifiers:

  • is-hoverable: The dropdown will show up when hovering the bu-dropdown.
  • is-active: The dropdown will show up all the time.
