Overview

@angular-bulma/components is a collection of easy to use, functional Angular components based around the components provided by the Bulma CSS framework. Each component is designed to be easy to use in any sized Angular project, repeatable, configurable, and implemented in a way such that boilerplate for the component is reduced from the traditional implementation in plain HTML.
