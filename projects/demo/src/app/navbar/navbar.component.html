Navbar

A responseive horizontal navbar that can support images,links, buttons, and dropdowns.

The bu-navbar component is a responsive and versitile horizontal navigation bar with the following structure:

  • bu-navbar: the main container.

    • div class="navbar-brand": the left side, always visible, which usually contains the logo and optionally some links of icons.

      Also contains a reactive burger on the right side which toggles the navbar menu on touch devices.

    • div class="navbar-start": The left part of the menu which appears next to the navbar brand on desktop.

    • div class="navbar-end": The right part of the menu which appears at the end of the navbar.

# Transparent navbar

To seamlessly integrate the navbar in any visual context, you can add the transparent attribute to the bu-navbar component. This will remove any hover or active background from the navbar items.

# Fixed navbar

You can now fix the navbar to either the top or bottom of the page. This is a 2-step process:

  • Add the fixed attribute to the bu-navbar component with a value of either top or bottom

                    
<bu-navbar fixed="top">

  • Add the corresponding has-navbar-fixed-top or has-navbar-fixed-bottom modifier to either the html or body element to provide the appropriate padding to the page.

    This should be done from within your app-root component.

                    
<body class="has-navbar-fixed-top">

  • danger

  • light

# Navbar Helper Classes

Additional helper attributes can be set on the bu-navbar:

  • spaced: Sets Topand Bottom paddings with 1rem, Left and Right paddings with 2rem
  • shadow: Adds a small amount of box-shadow around the bu-navbar
