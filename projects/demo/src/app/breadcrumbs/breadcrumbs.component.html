Breadcrumb

A simple breadcrumb component to improve a website's navigation experience.

The Bulma breadcrumb is a simple navigation component that only requires a bu-breadcrumbs container and a bu-breadcrumb list. The dividers are automatically created in the content of the ::before pseudo-element.

Additional attributes are available on the bu-breadcrumbs component:

  • alignment: Changes the alignment of the elements in the bu-breadcrumbs container. Can be set to center or left.

The is-active state of the bu-breadcrumb is controlled automatically by the bu-breadcrumbs component.

Example
@angular-bulma Components Documentation Breadcrumbs 
            
<bu-breadcrumbs>
  <bu-breadcrumb>
    <a href="#">@angular-bulma</a>
  </bu-breadcrumb>
  <bu-breadcrumb>
    <a href="#">Components</a>
  </bu-breadcrumb>
  <bu-breadcrumb>
    <a href="#">Documentation</a>
  </bu-breadcrumb>
  <bu-breadcrumb>
    <a href="#">Breadcrumbs</a>
  </bu-breadcrumb>
</bu-breadcrumbs>

# Alignment

For alternative alignments, use the alignment attribute on the bu-breadcrumbs element.

Alignments are centered and right

Example
@angular-bulma Components Documentation Breadcrumbs 
            
<bu-breadcrumbs alignment="centered">
  <bu-breadcrumb>
    <a href="#">@angular-bulma</a>
  </bu-breadcrumb>
  <bu-breadcrumb>
    <a href="#">Components</a>
